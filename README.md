# Styler Syntax Highlighter

A VS Code extension for custom Styler syntax highlighting.

## Features
- Syntax highlighting for `.styler` and `.stylerc` files

## Usage
Install the extension and open any Styler file to see custom syntax highlighting.

## Development
This extension uses a TextMate grammar for Styler. Edit `syntaxes/styler.tmLanguage.json` to customize highlighting rules.
